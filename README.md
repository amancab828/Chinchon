# Juego de Chinchón en Java
Proyecto desarrollado en Java que implementa el juego de cartas Chinchón, permitiendo partidas con jugadores humanos y controlados por IA.

---
# Características
- Soporte para 2 a 5 jugadores
- Jugadores humanos e IA
- Sistema de rondas y puntuación
- Detección automática de combinaciones (escaleras y tríos)
- Posibilidad de cerrar ronda
